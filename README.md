# nasa-apod

Nasa Apod is an application created as a take home exercise as part of an interview for [Obvious](https://obvious.in/)

## Download

You can download the APK from this [link](https://github.com/mddanishansari/nasa-apod/blob/main/release/app-debug.apk)

## Technologies

Various technlogies used in this project are as follows

- [Kotlin](https://kotlinlang.org/) programming language
- [MVVM](https://developer.android.com/jetpack/guide) architecture
- [Jetpack Components](https://developer.android.com/jetpack/) for the UI
- [ViewPager2](https://developer.android.com/jetpack/androidx/releases/viewpager2) for the slider
- [Coil](https://coil-kt.github.io/coil/) for loading images
- [Hilt](https://developer.android.com/training/dependency-injection/hilt-android) for dependency injection
- [RxJava](https://github.com/ReactiveX/RxJava) & [RxAndroid](https://github.com/ReactiveX/RxAndroid) for reactive stream and threading
- [Navigation Component](https://developer.android.com/guide/navigation) for navigating between screens
- [Jackson](https://github.com/FasterXML/jackson) for Serializing & Deserializing Json
